
# PGA Analytics 2026

A premium, deep-dive analytics dashboard for the 2026 PGA Tour season. Built with React, Vite, and Recharts.

## Features

- **Dashboard**: High-level overview of the season, featuring the World No. 1 and top leaders.
- **Deep Analytics**: detailed drill-downs for players comparing 2026 stats vs 2025 (Scoring Average, Driving Distance, SG: Total).
- **Tour Schedule**: Track upcoming and completed tournaments with live-updated status.
- **Rankings**: Official World Golf Ranking top 50 (simulated/mocked for demo).
- **Net Worth**: Financial insights and estimated net worth of top pros.

## Tech Stack

- **Core**: React + Vite
- **Styling**: Vanilla CSS (Premium Dark Mode + Glassmorphism)
- **Charts**: Recharts
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Locally**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## deployment

To deploy to GitHub Pages, updated `vite.config.js` with your base URL and run the build script.
